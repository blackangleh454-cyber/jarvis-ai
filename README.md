## 🎯 What is J.A.R.V.I.S?

J.A.R.V.I.S is a personal AI agent built for Linux that:

- 🎤 **Listens & Speaks** — Voice interaction powered by LiveKit + Google Gemini
- 🧠 **Learns & Remembers** — Infinite memory system that never forgets
- 🔒 **Secures Your System** — Built-in cybersecurity & privacy tools
- 🖥️ **Controls Everything** — System management, automation, monitoring
- 👁️ **Sees & Understands** — Computer vision & face recognition
- 🔧 **Self-Heals** — Auto-detects and fixes system issues
- 🚀 **Auto-Improves** — Learns from failures and builds its own skills

## 🛠️ Installation

### Prerequisites

```bash
# System dependencies
sudo apt update
sudo apt install -y python3 python3-pip ffmpeg xdotool xrandr git

# Optional for full features
sudo apt install -y v4l-utils cheese
```

### Setup

```bash
# Clone the repository
git clone https://github.com/blackangleh454-cyber/jarvis-ai.git
cd jarvis-ai

# Create virtual environment (recommended)
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Setup configuration
cp .env.example .env
# Edit .env with your API keys (see Configuration section)

# Run the installer
chmod +x setup_system.sh
./setup_system.sh
```

### Quick Start

```bash
# Activate environment
source venv/bin/activate

# Run J.A.R.V.I.S
python3 agent.py
```

## ⚙️ Configuration

Create a `.env` file with your API keys:

```env
# LiveKit (Required for voice)
LIVEKIT_URL=wss://your-livekit-server.livekit.cloud
LIVEKIT_API_KEY=your_api_key
LIVEKIT_API_SECRET=your_api_secret

# Google Gemini (Required for AI)
GOOGLE_API_KEY=your_gemini_api_key

# Optional: Google Search
GOOGLE_SEARCH_API_KEY=your_search_api_key
SEARCH_ENGINE_ID=your_search_engine_id

# Weather (Optional)
OPENWEATHER_API_KEY=your_weather_api_key
```

## 🎤 Usage

### Voice Commands

J.A.R.V.I.S responds to natural language:

```
"Open Firefox"
"Create a dark portfolio website called John"
"Remember my password is 12345"
"Search for Python tutorials"
"Take a screenshot"
"Check CPU usage"
"Open your eyes" (enable camera)
"What do you see?"
"Create folder called Projects"
"Download this file"
"Check system health"
```

### Text Commands

You can also type directly in the chat interface:

```
check_tool nmap
install_tool nmap
remember user name Mirza
recall projects
list_skills
check_system_health
```

### Available Commands

```bash
# Core commands
python3 agent.py              # Start J.A.R.V.I.S
python3 jarvis_daemon.py      # Run as background daemon

# System commands
./setup_system.sh             # Setup system integration
./jarvis-install              # Install system-wide
```

## 🔧 System Integration

### Desktop Entry

J.A.R.V.I.S can be installed as a desktop application:

```bash
# Install desktop entry
./jarvis-install
```

### Systemd Service (Background)

Run as a background service:

```bash
# Copy service file
sudo cp jarvis.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable jarvis
sudo systemctl start jarvis
```

## 🧠 How J.A.R.V.I.S Learns

### 1. From Failures
When a command fails, JARVIS:
- Records the failure pattern
- Detects recurring issues
- Auto-builds skills to fix them

### 2. From User Corrections
When you correct JARVIS:
- Logs the correction
- Updates behavior rules
- Never repeats the mistake

### 3. From Context
Every interaction:
- Saved to infinite memory
- Cross-referenced with past
- Used to improve future responses

### 4. Autonomous Gap Detection
Every 5 conversation turns:
- Analyzes failure patterns
- Identifies missing capabilities
- Auto-generates solution skills
