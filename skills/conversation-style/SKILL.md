# conversation-style

Jarvis talks like a partner.

## Commands

```bash
python3 handler.py style                     # Current conversation style
python3 handler.py set <style>              # Set conversation style
python3 handler.py formal                   # Switch to formal
python3 handler.py casual                   # Switch to casual
python3 handler.py technical               # Switch to technical
python3 handler.py friendly                # Switch to friendly
```
