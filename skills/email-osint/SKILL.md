# Email OSINT

**Description:** Gather publicly available info about email addresses - breach checking, username discovery, social media presence.

**Commands:**
- `lookup <email>` - Full email analysis
- `breach <email>` - Check for breaches
- `username <email>` - Extract username
- `domain <email>` - Get email domain info
- `exist <email>` - Check if email exists

**Usage:**
```bash
python handler.py lookup user@example.com
python handler.py breach test@gmail.com
```
