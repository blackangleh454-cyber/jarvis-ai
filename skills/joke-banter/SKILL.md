# joke-banter

Jarvis has personality.

## Commands

```bash
python3 handler.py joke                         # Tell a joke
python3 handler.py banter <topic>            # Casual banter
python3 handler.py roast <subject>            # Friendly roast
python3 handler.py compliment                  # Give a compliment
python3 handler.py reaction <emotion>         # React emotionally
python3 handler.py pun                           # Tell a pun
```
