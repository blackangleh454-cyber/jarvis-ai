# learning-engine

Jarvis learns from your corrections.

## Commands

```bash
python3 handler.py correct "<issue>" "<fix>"     # Record correction
python3 handler.py feedback "<type>" "<note>"  # Give feedback
python3 handler.py insights                     # Learning insights
python3 handler.py patterns                    # Behavioral patterns
python3 handler.py apply                       # Apply learned fixes
```
