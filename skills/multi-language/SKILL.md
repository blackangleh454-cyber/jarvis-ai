# multi-language

Jarvis understands mixed languages.

## Commands

```bash
python3 handler.py detect <text>             # Detect language
python3 handler.py translate <text> <to>   # Translate text
python3 handler.py switch <lang>            # Switch interface language
python3 handler.py status                   # Language settings
```
