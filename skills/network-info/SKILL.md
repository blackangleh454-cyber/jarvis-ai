# Network Info

**Description:** Comprehensive network information - interfaces, IPs, routes, connections, DNS, and more.

**Commands:**
- `interfaces` - Show all network interfaces
- `ip` - Show IP addresses
- `gateway` - Show default gateway
- `dns` - Show DNS servers
- `routes` - Show routing table
- `connections` - Show active connections
- `stats` - Show network statistics
- `wifi` - Show WiFi info
- `all` - Full network summary

**Usage:**
```bash
python handler.py all
python handler.py interfaces
python handler.py wifi
python handler.py connections
```
