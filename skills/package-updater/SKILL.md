# Smart Package Updater

**Description:** Intelligent package updates - apt, flatpak, snap with smart filtering and safety checks.

**Commands:**
- `check` - Check for updates
- `update` - Update package lists
- `upgrade` - Upgrade all packages
- `upgrade-safe` - Safe upgrade (keep config)
- `upgrade-dist` - Distribution upgrade
- `clean` - Clean up
- `autoremove` - Remove unused packages
- `history` - Show upgrade history
- `check-flatpak` - Check flatpak updates
- `check-snap` - Check snap updates
- `upgrade-all` - Upgrade everything

**Usage:**
```bash
python handler.py check
python handler.py upgrade-safe
python handler.py upgrade-all
```
