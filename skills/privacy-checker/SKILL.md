# Privacy & Leak Checker

**Description:** Deep privacy analysis - check for IP leaks, DNS leaks, WebRTC leaks, VPN detection, and privacy score.

**Commands:**
- `check` - Full privacy check
- `my-ip` - Show your IP address
- `check-dns` - Check for DNS leaks
- `check-webrtc` - Check WebRTC leaks
- `check-vpn` - Detect VPN
- `check-ipv6` - Check IPv6 leaks
- `check-headers` - Check HTTP headers
- `score` - Calculate privacy score
- `test-speed` - Internet speed test
- `dns-servers` - Show DNS history

**Usage:**
```bash
python handler.py check
python handler.py my-ip
python handler.py check-dns
python handler.py score
```
