# System Health Doctor

**Description:** Intelligent system health diagnostics - detects issues, analyzes performance, and provides human-like recommendations.

**Commands:**
- `diagnose` - Full system health check
- `check cpu` - CPU health analysis
- `check memory` - RAM health analysis  
- `check disk` - Disk health analysis
- `check network` - Network diagnostics
- `check security` - Security audit
- `check services` - Critical services status
- `quick` - Quick health summary
- `suggest` - AI-like recommendations

**Usage:**
```bash
python handler.py diagnose
python handler.py quick
python handler.py suggest
python handler.py check disk
```
