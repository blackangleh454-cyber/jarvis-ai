# YouTube Summary

**Description:** Get AI-powered summaries of YouTube videos using Gemini AI.

**Commands:**
- `summarize <url>` - Get video summary
- `summary <url>` - Get detailed summary
- `quick <url>` - Quick 1-paragraph summary
- `transcript <url>` - Get video transcript
- `chapters <url>` - List video chapters

**Usage:**
```bash
python handler.py summarize "https://youtube.com/watch?v=..."
python handler.py transcript "https://youtube.com/watch?v=..."
```
